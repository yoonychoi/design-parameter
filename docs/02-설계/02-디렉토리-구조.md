## 2. 디렉토리별 책임

| 디렉토리 | 책임 | 의존 방향 |
|----------|------|-----------|
| `types/` | 타입 정의 | 의존 없음 (최하위) |
| `constants/` | 참조 테이블, 상수 | types에만 의존 |
| `calculations/` | 경험식 계산 (순수 함수) | types, constants에만 의존 |
| `state/` | 전역 상태 관리 | types, calculations에 의존 |
| `components/common/` | 재사용 UI 컴포넌트 | types에만 의존 |
| `components/tabs/` | 탭별 페이지 | state, common, calculations에 의존 |
| `export/` | 엑셀 내보내기 | types, state에 의존 |

---

## 3. 의존성 규칙

```
types  ←  constants  ←  calculations  ←  state  ←  components/tabs
  ↑                                        ↑            ↑
  └──────── components/common ─────────────┘            │
                                                        │
                                          export ───────┘
```

- `calculations/`는 React에 의존하지 않는다 (순수 TypeScript)
- `components/common/`은 비즈니스 로직을 포함하지 않는다
- `components/tabs/`만 state와 calculations를 함께 사용한다
