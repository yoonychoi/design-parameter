## 8. 공통 적용 규칙

| ID | 규칙 | 설명 |
|----|------|------|
| BR-CM-01 | 선행 적용값 | 상위 모듈의 적용값이 하위 모듈에 자동 반영된다 |
| BR-CM-02 | 재계산 | 상위 모듈에서 값 변경 시 하위 모듈이 자동 재계산된다 |
| BR-CM-03 | 적용 옵션 기본값 | 시험값/인근자료가 없으면 경험식 결과를 기본 적용한다 |
| BR-CM-04 | 적용 옵션 직접입력 | 시험값/인근자료가 있거나 필요시 직접 입력한다 |
| BR-CM-05 | 공란 처리 | 조건 미충족 시 해당 경험식은 공란으로 표시하고, 평균/범위 산출에서 제외한다 |
