---

## 2. 화면 구성 방식

- **네비게이션**: 상단 탭 바 (8개 탭)
- **순차 진행 권장**, 탭 간 자유 이동 가능
- **하단 버튼**: [이전] [다음] [엑셀 다운로드]

---

## 3. 화면별 주요 구성 요소

### SCR-01. N값 분포

| 영역 | 구성 요소 |
|------|-----------|
| 프로젝트 설정 | 지층 추가/삭제, 시추공 추가/삭제 |
| N값 입력 | 시추공 선택 드롭다운, 동적 테이블 (심도/지층/N값), 행 추가/삭제 |
| 산출 결과 | 지층별 요약 테이블 (지층명, 측정치 수, 대표N값) |
| 그래프 | 심도별 N값 분포 차트 (전체 시추공 오버레이) |

### SCR-02. 내부마찰각

| 영역 | 구성 요소 |
|------|-----------|
| 산정 테이블 | 지층명(자동), 대표N값(자동), 적용N값(입력), 경험식 5개(자동), 평균(자동), 범위(자동), 시험값(입력), 인근자료(입력), 적용(선택) |

### SCR-03. 점착력

| 영역 | 구성 요소 |
|------|-----------|
| 산정 테이블 | 지층명(자동), 대표N값(자동), 적용N값(자동), 경험식 3개(자동), 평균(자동), 범위(자동), 시험값(자동), 인근자료(입력), 적용(선택) |

### SCR-04. 변형계수

| 영역 | 구성 요소 |
|------|-----------|
| 산정 테이블 | 지층명(자동), 적용N값(자동), 지층구분(입력), Schmertmann α(입력), Bowles 입력치(선택), 경험식 6개(자동/조건부), 평균(자동), 범위(자동), 시험값(입력), 인근자료(입력), 적용(선택) |
| 참조자료 | Schmertmann α 계수표 팝오버, Bowles 조건식 팝오버 |

### SCR-05. 수평지반반력계수

| 영역 | 구성 요소 |
|------|-----------|
| 산정 테이블 | 지층명(자동), 적용N값(자동), 지층구분(자동), 점착력(자동), 경험식 산정값(자동), 적용(선택) |

### SCR-06. 앵커주면마찰저항

| 영역 | 구성 요소 |
|------|-----------|
| 산정 테이블 | 지층명(자동), 적용N값(자동), 지층구분(자동), 지반 종류, 산정값 |
| 암반층 추가 | 별도 지층 추가 입력 영역 |

### SCR-07. 투수계수

| 영역 | 구성 요소 |
|------|-----------|
| 산정 테이블 | 지층명(자동), 지층구분(자동), D10(입력), D20(입력), Hazen(자동), Creager(자동), 현장시험값(입력), 인근자료(입력), 적용(선택), 비고(자동) |
| 참조자료 | Creager 테이블 팝오버, USBR 테이블 팝오버 |

### SCR-08. 최종 요약

| 영역 | 구성 요소 |
|------|-----------|
| 요약 테이블 | 지층명, 대표N값, 적용N값, 단위중량(입력), 점착력, 내부마찰각, 변형계수, 수평지반반력계수, 투수계수 |
| 엑셀 다운로드 | 다운로드 버튼 (요약 + 모듈별 상세 시트) |
