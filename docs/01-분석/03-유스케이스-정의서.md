## 3. 유스케이스 상세

### UC-01. 프로젝트 설정

**주 액터:** 사용자

**사전조건:** 없음

**기본 흐름:**
1. 사용자가 지층명을 입력하고 추가 버튼을 클릭한다
2. 시스템이 지층 목록에 추가한다
3. 사용자가 시추공 공번을 입력하고 추가 버튼을 클릭한다
4. 시스템이 시추공 목록에 추가한다
5. 필요에 따라 1~4를 반복한다

**대안 흐름:**
- 사용자가 기존 지층/시추공을 삭제할 수 있다
- 삭제 시 해당 지층/시추공에 연결된 N값 데이터도 함께 삭제된다

**사후조건:** 지층 목록과 시추공 목록이 구성된다

---

### UC-02. N값 입력

**주 액터:** 사용자

**사전조건:** UC-01에서 지층과 시추공이 최소 1개 이상 등록됨

**기본 흐름:**
1. 사용자가 시추공을 선택한다
2. 사용자가 측정 행을 추가한다
3. 심도(순서), 지층, N값을 입력한다
4. 필요에 따라 2~3을 반복한다
5. 다른 시추공을 선택하여 1~4를 반복한다

**대안 흐름:**
- 측정 행을 삭제할 수 있다
- 공번별 측정 행 수는 독립적이다

**사후조건:** 시추공별 N값 데이터가 입력된다

---

### UC-03. N값 분포 확인

**주 액터:** 사용자

**사전조건:** UC-02에서 N값 데이터가 입력됨

**기본 흐름:**
1. 시스템이 지층별 측정치 수와 대표 N값(평균)을 자동 산출한다
2. 시스템이 심도별 N값 분포 그래프를 표시한다
3. 사용자가 결과를 확인한다

**사후조건:** 지층별 대표N값이 후속 모듈에 전달 가능한 상태가 된다

---

### UC-04. 강도정수 산정

**주 액터:** 사용자

**사전조건:** UC-03에서 대표N값이 산출됨

**기본 흐름 (내부마찰각):**
1. 시스템이 지층명, 대표N값을 자동 표시한다
2. 사용자가 필요 시 적용N값을 수정한다
3. 시스템이 5개 경험식 결과, 평균, 범위를 자동 산출한다
4. 사용자가 시험값 또는 인근자료를 입력한다 (선택)
5. 사용자가 적용 옵션을 선택한다 (경험식 평균 / 직접 입력)

**기본 흐름 (점착력):**
1. 시스템이 선행 적용값(지층명, 대표N값, 적용N값)을 자동 표시한다
2. 시스템이 3개 경험식 결과, 평균, 범위를 자동 산출한다
3. 사용자가 시험값 또는 인근자료를 입력한다 (선택)
4. 사용자가 적용 옵션을 선택한다

**사후조건:** 내부마찰각, 점착력 적용값이 후속 모듈에 전달 가능한 상태가 된다

---

### UC-05. 변형계수 산정

**주 액터:** 사용자

**사전조건:** UC-04 완료

**기본 흐름:**
1. 시스템이 선행 적용값을 자동 표시한다
2. 사용자가 지층구분(점토/실트/모래/자갈)을 입력한다
3. 사용자가 Schmertmann α 계수를 입력한다 (선택, 참조자료 팝오버 확인 가능)
4. 사용자가 Bowles 입력치를 선택한다 (선택, 참조자료 팝오버 확인 가능)
5. 시스템이 6개 경험식 결과를 산출한다 (조건 미충족 시 공란)
6. 시스템이 공란 제외 평균, 범위를 산출한다
7. 사용자가 적용 옵션을 선택한다

**사후조건:** 변형계수 적용값이 후속 모듈에 전달 가능한 상태가 된다

---

### UC-06. 수평지반반력계수 산정

**주 액터:** 사용자

**사전조건:** UC-05 완료

**기본 흐름:**
1. 시스템이 선행 적용값(지층명, 적용N값, 지층구분, 점착력)을 자동 표시한다
2. 시스템이 지층구분에 따라 Hukuoka 또는 Soletanche 경험식을 적용한다
3. 사용자가 적용 옵션을 선택한다 (경험식 / 직접 입력)

**사후조건:** 수평지반반력계수 적용값 확정

---

### UC-07. 앵커주면마찰저항 산정

**주 액터:** 사용자

**사전조건:** UC-06 완료

**기본 흐름:**
1. 시스템이 선행 적용값을 자동 표시한다
2. 암반층인 경우 사용자가 별도 지층을 추가한다
3. 시스템이 지반 종류 및 N값 기반으로 산정한다

**사후조건:** 앵커주면마찰저항 적용값 확정

---

### UC-08. 투수계수 산정

**주 액터:** 사용자

**사전조건:** UC-07 완료

**기본 흐름:**
1. 시스템이 선행 적용값(지층명, 지층구분)을 자동 표시한다
2. 사용자가 D10, D20 값을 입력한다
3. 시스템이 Hazen(D10 입력 시), Creager(D20 입력 시) 결과를 산출한다
4. 사용자가 현장시험값, 인근자료를 입력한다 (선택)
5. 사용자가 적용 옵션을 선택한다
6. 시스템이 비고란에 적용 근거를 자동 표시한다

**사후조건:** 투수계수 적용값 확정

---

### UC-09. 최종 요약 확인

**주 액터:** 사용자

**사전조건:** UC-02~UC-08의 결과가 존재

**기본 흐름:**
1. 시스템이 전체 지층의 산정 결과를 통합 테이블로 표시한다
2. 사용자가 단위중량(rt)을 직접 입력한다
3. 사용자가 결과를 검토한다

**사후조건:** 전체 설계지반정수 확정

---

### UC-10. 엑셀 다운로드

**주 액터:** 사용자

**사전조건:** UC-09 완료

**기본 흐름:**
1. 사용자가 엑셀 다운로드 버튼을 클릭한다
2. 시스템이 요약 시트 + 각 모듈 상세 시트를 포함한 엑셀 파일을 생성한다
3. 브라우저가 파일 다운로드를 시작한다

**사후조건:** 엑셀 파일 다운로드 완료
