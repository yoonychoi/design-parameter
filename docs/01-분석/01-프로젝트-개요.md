# 01. 프로젝트 개요

> 작성일: 2026-02-19

---

## 1. 프로젝트 배경

지반조사 결과(표준관입시험 N값)를 바탕으로 설계지반정수를 산정하는 작업은 지반공학 실무에서 반복적으로 수행된다. 현재 이 작업은 엑셀 기반으로 이루어지고 있으며, 다음과 같은 한계가 있다.

- 시추공 수와 지층 구성이 프로젝트마다 달라 매번 엑셀 양식을 수정해야 함
- 경험식 간 조건 분기(공란 처리, 지층구분별 적용 등)가 복잡하여 오류 발생 가능
- 모듈 간 데이터 연계(선행 적용값 반영)가 수식 참조로 관리되어 유지보수 어려움

## 2. 프로젝트 목적

엑셀 기반 작업을 웹 애플리케이션으로 전환하여:

1. **가변적인 입력 구조** (시추공 수, 측정값 수) 를 유연하게 처리
2. **경험식 자동 계산** 및 조건 분기를 정확하게 적용
3. **모듈 간 데이터 자동 연계** 로 일관성 확보
4. **결과를 엑셀로 내보내기** 하여 기존 업무 흐름과 호환

## 3. 프로젝트 범위

### 3.1 범위 내 (In Scope)

| 구분 | 내용 |
|------|------|
| N값 분포 | 시추공별 N값 입력, 지층별 대표N값 산출, 분포 그래프 |
| 내부마찰각 | Dunham, Peck, Meyerhof, Ohsaki, 도로교시방서 경험식 |
| 점착력 | Dunham, Terzaghi-Peck, Ohsaki 경험식 |
| 변형계수 | Schmertmann, Bowles, Yoshinaka, Hisatake, 도로교시방서, 구조물기초설계기준 |
| 수평지반반력계수 | Hukuoka(모래/자갈), Soletanche(점토/실트) |
| 앵커주면마찰저항 | 지반 종류 및 N값 기반 산정 |
| 투수계수 | Hazen, Creager 경험식 + 참조테이블 |
| 최종 요약 | 전체 결과 통합 테이블 + 단위중량 직접 입력 |
| 결과 출력 | 화면 표시 + 엑셀 다운로드 |

### 3.2 범위 외 (Out of Scope)

- 사용자 인증 및 권한 관리
- 프로젝트 저장/불러오기 (DB 저장)
- 엑셀 파일 업로드를 통한 입력
- PDF 보고서 생성
