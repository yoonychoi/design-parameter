# 02. 요구사항 정의서

> 작성일: 2026-02-19

---

## 1. 기능 요구사항

### FR-01. N값 분포 입력 및 산출

| ID | 요구사항 | 우선순위 |
|----|----------|----------|
| FR-01-01 | 지층 목록을 자유롭게 추가/삭제할 수 있다 | 필수 |
| FR-01-02 | 시추공(공번)을 자유롭게 추가/삭제할 수 있다 | 필수 |
| FR-01-03 | 시추공별로 심도/지층/N값을 입력할 수 있으며, 측정 행 수는 가변적이다 | 필수 |
| FR-01-04 | 지층별 대표 N값(평균)을 자동 산출한다 | 필수 |
| FR-01-05 | 지층별 측정치 수를 자동 산출한다 | 필수 |
| FR-01-06 | 심도별 N값 분포 그래프를 표시한다 (전체 시추공 오버레이) | 필수 |

### FR-02. 내부마찰각 산정

| ID | 요구사항 | 우선순위 |
|----|----------|----------|
| FR-02-01 | 지층명, 대표N값은 N값 분포 결과에서 자동 반영한다 | 필수 |
| FR-02-02 | 적용 N값을 직접 입력할 수 있다 (기본값: 대표N값) | 필수 |
| FR-02-03 | Dunham, Peck, Meyerhof, Ohsaki, 도로교시방서 5개 경험식으로 산정한다 | 필수 |
| FR-02-04 | Meyerhof는 N < 10인 경우 공란(미산정) 처리한다 | 필수 |
| FR-02-05 | 산정된 경험식의 평균(소수점 절사)과 범위를 표시한다 | 필수 |
| FR-02-06 | 시험값, 인근자료를 직접 입력할 수 있다 | 필수 |
| FR-02-07 | 적용값을 2가지 옵션으로 선택한다: 경험식 평균 / 직접 입력 | 필수 |

### FR-03. 점착력 산정

| ID | 요구사항 | 우선순위 |
|----|----------|----------|
| FR-03-01 | 지층명, 대표N값, 적용N값은 선행 적용값을 자동 반영한다 | 필수 |
| FR-03-02 | Dunham, Terzaghi-Peck, Ohsaki 3개 경험식으로 qu를 산정하고 c = qu/2로 환산한다 | 필수 |
| FR-03-03 | 평균, 범위, 시험값, 인근자료, 적용 옵션은 내부마찰각과 동일 패턴이다 | 필수 |

### FR-04. 변형계수 산정

| ID | 요구사항 | 우선순위 |
|----|----------|----------|
| FR-04-01 | 지층명, 적용N값은 선행 적용값을 자동 반영한다 | 필수 |
| FR-04-02 | 지층구분(점토/실트/모래/자갈)을 직접 입력한다 | 필수 |
| FR-04-03 | Schmertmann α 계수를 직접 입력한다 (참조자료 팝오버 제공) | 필수 |
| FR-04-04 | Bowles 입력치를 직접 선택한다 (참조자료 팝오버 제공) | 필수 |
| FR-04-05 | Schmertmann: α 미입력 시 공란, Bowles: 입력치 미선택 시 공란 | 필수 |
| FR-04-06 | 구조물기초설계기준: 모래/자갈만 산정, 나머지 공란 | 필수 |
| FR-04-07 | 평균/범위는 공란 제외한 값들로 산출한다 | 필수 |
| FR-04-08 | 시험값, 인근자료, 적용 옵션 동일 패턴 | 필수 |

### FR-05. 수평지반반력계수 산정

| ID | 요구사항 | 우선순위 |
|----|----------|----------|
| FR-05-01 | 지층명, 적용N값, 지층구분, 점착력은 선행 적용값을 자동 반영한다 | 필수 |
| FR-05-02 | 모래/자갈층: Hukuoka, 점토/실트층: Soletanche 경험식으로 산정한다 | 필수 |
| FR-05-03 | 적용: 경험식 산정값 / 직접 입력 선택 | 필수 |

### FR-06. 앵커주면마찰저항 산정

| ID | 요구사항 | 우선순위 |
|----|----------|----------|
| FR-06-01 | 지층명, 적용N값, 지층구분은 선행 적용값을 자동 반영한다 | 필수 |
| FR-06-02 | 암반층인 경우 별도 추가 지층을 고려할 수 있다 | 필수 |
| FR-06-03 | 지반 종류 및 N값 기반으로 앵커주면마찰저항을 산정한다 | 필수 |

### FR-07. 투수계수 산정

| ID | 요구사항 | 우선순위 |
|----|----------|----------|
| FR-07-01 | 지층명, 지층구분은 선행 적용값을 자동 반영한다 | 필수 |
| FR-07-02 | D10(mm), D20(mm)을 직접 입력한다 | 필수 |
| FR-07-03 | Hazen: k = 1.5 × D10², D10 입력 시 산정 | 필수 |
| FR-07-04 | Creager: D20 → 참조테이블 조회, D20 입력 시 산정 | 필수 |
| FR-07-05 | 현장시험값, 인근자료를 직접 입력할 수 있다 | 필수 |
| FR-07-06 | 적용: 경험식 최대값 / 직접 입력 선택 | 필수 |
| FR-07-07 | 비고란에 적용 근거를 자동 표시한다 | 필수 |
| FR-07-08 | Creager 참조테이블(D20→Ks), USBR 참조테이블(USCS→Ks)을 팝오버로 제공한다 | 필수 |

### FR-08. 최종 요약

| ID | 요구사항 | 우선순위 |
|----|----------|----------|
| FR-08-01 | 전체 지층의 산정 결과를 한 화면에 통합 표시한다 | 필수 |
| FR-08-02 | 단위중량(rt, kN/m³)을 직접 입력할 수 있다 | 필수 |
| FR-08-03 | 엑셀 다운로드: 요약 시트 + 각 모듈 상세를 시트별로 내보낸다 | 필수 |

## 2. 비기능 요구사항

| ID | 요구사항 | 우선순위 |
|----|----------|----------|
| NFR-01 | 모든 계산은 브라우저에서 수행한다 (서버 불필요) | 필수 |
| NFR-02 | 세션 기반으로 동작하며, 별도 DB를 사용하지 않는다 | 필수 |
| NFR-03 | 인증 없이 바로 사용 가능하다 | 필수 |
| NFR-04 | 값 변경 시 하위 모듈이 즉시 재계산된다 | 필수 |
| NFR-05 | 정적 호스팅으로 배포 가능하다 (Vercel, Netlify, GitHub Pages 등) | 필수 |
| NFR-06 | 주요 브라우저(Chrome, Edge, Firefox)에서 정상 동작한다 | 필수 |
