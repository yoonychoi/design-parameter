# 설계지반정수 산정 프로그램 — 개발 결과 요약

> 작성일: 2026-02-20

---

## 1. 프로젝트 개요

지반 조사 결과(N값)를 입력하면 설계에 필요한 지반정수를 자동 산정하는 웹 애플리케이션.
기존 엑셀 작업을 웹 기반으로 대체하여 누구나 브라우저에서 접근 가능하도록 구현.

## 4. 구현 기능

### 산정 항목 (탭 구성)

| 탭 | 내용 |
|----|------|
| N값 분포 | 지층·시추공 입력, 대표 N값 산정(Rounddown), 지층별 분산형 그래프 |
| 내부마찰각 | Dunham / Peck / Meyerhof / Ohsaki / 도로교시방서 경험식 |
| 점착력 | Dunham / Terzaghi-Peck / Ohsaki (Cu=qu/2 환산) |
| 변형계수 | Schmertmann / Bowles / Yoshinaka / Hisatake / 도로교 / 구조물기초설계기준 |
| 수평지반반력계수 | 사질토: Hukuoka(6910·N^0.406) / 점성토: Soletanche(500·(10+Cu/4)) |
| 앵커주면마찰 | 지반 종류별 산정 + 경험값 참조표 |
| 투수계수 | Hazen(모래만) / Creager(D20 참조표) / 소수 셋째자리 지수 표기 |
| 최종 요약 | 단위중량 직접 입력, 전 항목 일람, 엑셀 다운로드 |

### 공통 기능

- 각 항목별 **경험식 / 직접입력** 선택 적용
- 시험값·인근자료 직접 입력 반영
- 전 탭 경험식 **각주 수식 표기**
- **저장(JSON)** / **불러오기(JSON)** — 작업 내용 보존 및 재사용
- **엑셀 다운로드** — 최종 요약 결과 출력
