# 설계지반정수 산정

## N값 분포

* 주상도 갯수
* 지층 갯수
* 지층명 / 측정치 수
* N값
* 그래프(지층별, 심도별 N값)

## 강도정수

### 내부마찰각

* 지층명
* 대표 N값(평균)
* 적용 N값(직접 입력)
* 경험식 : Dunham / Peck / Meyerhof / Ohsaki / 도로교시방서
* 평균
* 범위
* 시험값(직접 입력)
* 인근자료(직접 입력)
* 적용(2가지 선택옵션 적용, 1) 시험값 및 인근자료값이 없는 경우 경험식 평균값 반영,  2) 시험값 및 인근자료값이 있거나 필요시 직접 입력)

\###점착력(kPa)

* 지층명(선행 적용값 반영)
* 대표 N값(선행 적용값 반영)
* 적용 N값(선행 적용값 반영)
* 경험식 : Dunham / Terzahi-Peck / Ohsaki (Cu=qu/2 환산값 적용)
* 평균
* 범위
* 시험값(선행 적용값 반영)
* 인근자료(직접 입력)
* 적용(2가지 선택옵션 적용, 1) 시험값 및 인근자료값이 없는 경우 경험식 평균값 반영,  2) 시험값 및 인근자료값이 있거나 필요시 직접 입력)

\##변형계수(MPa)

* 지층명(선행 적용값 반영)
* 적용 N값(선행 적용값 반영)
* 지층구분(점토/ 실트/ 모래/ 자갈 구분, 직접입력)
* Schmertmann, α (실트 4/ 세립 또는 중립 모래 7/ 조립 모래 10/ 자갈 12, 직접 입력)
* Bowles 입력치(모래/ 점토질 모래/ 실트질 모래/ 자갈섞인 모래, 직접 입력)
* 경험식 : Schmertmann(α 입력값이 없으면 공란 처리) / Bowles(입력치가 없으면 공란 처리) / Yoshinaka / Hisatake / 도로교시방서 / 구조물기초설계기준(지층구분에서 모래, 자갈인 경우 산정, 모래, 자갈이 아니면 공란 처리)
* 평균
* 범위
* 시험값(직접 입력)
* 인근자료(직접 입력)
* 적용(2가지 선택옵션 적용, 1) 시험값 및 인근자료값이 없는 경우 경험식 평균값 반영,  2) 시험값 및 인근자료값이 있거나 필요시 직접 입력)

\##수평지반반력계수(kN/m^3)

* 지층명(선행 적용값 반영)
* 적용 N값(선행 적용값 반영)
* 지층구분(선행 적용값 반영)
* 점착력(선행 적용값 반영)
* 경험식 산정 (모래/ 자갈층인 경우 Hukuoka, 점토/ 실트층인 경우 Soletanche)
* 적용(2가지 선택옵션 적용, 1) 경험식 산정값 반영,  2) 직접 입력)

\##앵커주면마찰저항(kN/m^2)

* 지층명(선행 적용값 반영)
* 적용 N값(선행 적용값 반영)
* 지층구분(선행 적용값 반영, 암반층인 경우 별도 추가 지층 고려)
* 앵커주면마찰저항(kN/m^2) 산정

\##투수계수 산정

* 지층명(선행 적용값 반영)
* 지층구분(선행 적용값 반영)
* Soil Test Data(D10/ D20 입력칸 반영)
* 경험식(Hazen, Creager) 
* 현장시험값(직접입력)
* 인근자료(직접입력)
* 적용(2가지 선택옵션 적용, 1) 시험값 및 인근자료값이 없는 경우 경험식 최대값 반영,  2) 시험값 및 인근자료값이 있거나 필요시 직접 입력)



\## 보완이 필요한 부분 의견

1. 입력 데이터 구조 미정의

* &nbsp;엑셀의 N치 시트처럼 원시 데이터를 입력받을 수 있는 폼 제시

2\. 단위중량 누락

* 최종 산정 시트에 직접 입력 가능한 폼 제공

3\. 변형계수 

* 알파계수를 직접입력 가능하도록 참조자료 제공
* Bowles 조건식을 직접 선택할 수 있도록 참조자료 제공
* 구조물기초설계기준은 모래/ 자갈층만 산정, 나머지는 공란으로 표현

4\. N값 50타 초과 처리 규칙

* 무시, 사용자의 원시 데이터 입력값을 반영

5\. 최종 요약 테이블

* 엑셀 시트처럼 모든 결과를 한 화면에 보여주는 요약 뷰 제공



\##보완이 필요한 부분 의견2

1. 내부마찰각 경험식 오류

* Peck 마찰각=0.3N+27
* &nbsp;Meyerhof =0.25N+32.5 (단, N값이 10이상인 경우 산정, 10보다 작으면 공란-미산정)
* &nbsp;도로교시방서 = sqrt(15N)+15

2\. 점착력 경험식 오류

* Dunham qu=N/0.077
* Terzaghi-Peck qu=N/0.082
* Ohsaki qu=40+N/0.2

3\. 변형계수 경험식 오류

* Yoshinaka E=0.678\*N^0.993



\##보완의견3

* 대표 N값 평균값은 Rounddown 적용
* 심도별 N값 분포 그래프 오류 : 심도별 N값을 plot하지 못함
* 내부마찰각 경험식 수식 표현 : 예를들어 Dunham φ=sqrt(12N)+15 등, 표 아래에 각주로 표현 등 표현방법을 알아서 할 것
* 점착력은 위에 타이틀을 점착력(kPa)   Cu=qu/2로 수정 (- 표현도 삭제)
* 점착력 경험식도 수식 표현 :  예를들어 Dunham qu=N/0.077, 표 아래에 각주로 표현 등 표현방법은 내부마찰각과 동일하게 할 것
* 변형계수 경험식도 각주로 표현 Schmertmann E=αN(0.1 MPa), Yoshinaka, Hisatake, 구조묽ㅣ초설계기준, 도로교 등
* 수평지반반력계수 경험식 오류, "자갈", "모래"와 같은 사질토는 Hukuoka식 적용 6910\*N^0.406, "점토", "실트"와 같은 점성토는 Soletanche 적용 500\*(10+Cu/4)
* 수평지반반력계수도 표 아래에 각주로 수식 표현
* 앵커주면마찰 표 아래에 경험값 표 참조자료로 삽입
* 투수계수 산정시 "모래" 조건만 Hazen식 반영, 표 아래에 Hazen식 K=C(D10)^2 cm/s 각주 표현
* 투수계수 산정값은 모두 소수 셋째자리까지 지수함수로 표현. 예를들어 4.500E-02



\##보완의견4

* 내부마찰각 적용에서 직접입력이 되지 않음. 직접입력 선택 후 값이 입력 가능하도록 수정
* 현재 내부마찰각 적용 직접입력 후 타이핑 시 적용N값이 수정됨.  적용N값은 N값이고 적용은 사용자가 최종 결정한 내부마찰각임
* 점착력 적용 직접입력 타이핑 안됨
* 변형계수 적용 직접입력 타이핑 안됨
* 투수계수 Creager 산정시 D20에 해당하는 ks값이 없는 경우 바로 위 입경의 ks값 반영, 최대값은 1.8 적용
* 투수계수 현장시험 및 인근자료 입력시 표시는 소수 셋째자리 지수로 표기
* 투수계수 각주 산정값 표기 내용 줄은 삭제
* 투수계수 적용 직접입력 타이핑 안됨
* 설계지반정수 요약 투수계수는 소수 셋째자리 지수로 표기
